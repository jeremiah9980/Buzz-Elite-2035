Import NCS Games

Build a safe local command that reads the NCS schedule with your logged-in Chrome session, previews upcoming games, and then adds them to the selected GameChanger team.

Build Import Command

Paste the exact NCS schedule page and the GameChanger team schedule page.

Local Command

Enter both schedule URLs, then choose Dry Run or Live Import.
Start with Dry Run. Review the games written to ncs-monitor/reports/ncs-gamechanger-import.json. Only use Live Import after the dates, opponents, times, and locations look correct.

Recommended Flow

  1. Pull the latest repository changes.
  2. Install the local browser dependencies.
  3. Close Chrome so the profile can be cloned cleanly.
  4. Run the generated dry-run command.
  5. Review the extracted schedule manifest.
  6. Run the live command with --confirm.
  7. Refresh the GameChanger schedule widget on the portal.
